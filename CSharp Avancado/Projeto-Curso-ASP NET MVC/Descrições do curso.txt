ASP .NET MVC na Prática - Aula parte 7 (Criando Ares)

* Criando área administrativa
	-> Para isto é necessário clicar com o botão direito em cima do nosso projeto e Add Area.
	-> Após criar a área, para acessar a áres pela url é preciso digitar o caminho completo para conseguir acessar: https://site.com/Admin/Home/Index
	-> Para corrigir é necessário informar no RouteConfig o namespaces: new [] { "NAMESPACE DO PROJETO"}
	-> Em seguida, acessar o arquivo AdminAreaRegistratio para e adicionar um nome de controle para a rota ficar correta.
	-> Para manter o layout das páginas na área administrativa, na pagina index, abaixo do titulo informe o Layout: "CAMINHO FISICO DO ARQUIVO"
